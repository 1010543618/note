# cmd命令

## windows服务相关
### net
- 启动`net start ServiceName`
- 停止`net stop ServiceName`
- 显示用法`net`

### sc
- 启动`sc start ServiceName`
- 停止`sc stop ServiceName`
- 显示用法`sc`

## 文件操作
1. 查看文件：dir  
1. 创建文件: >filename  
1. 写文件: test>filename
