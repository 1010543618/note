# Q1：富文本编辑器在模态框中只有第一次有效

解决：在关闭模态框时移除富文本编辑器

Kindeditor 解决方法：

    $('#myModal').on('hidden.bs.modal', function () {
        // 关闭Dialog前移除编辑器
        KindEditor.remove('#content');
    });

UEditor 解决方法：

    $('#adminModalLg').on('hidden.bs.modal', function () {
        // 关闭Dialog前移除编辑器
        UM.getEditor('container').destroy();
    });
