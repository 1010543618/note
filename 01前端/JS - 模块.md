# CommonJS

-   [CommonJS - Wikipedia](https://en.wikipedia.org/wiki/CommonJS)

## 介绍

主要在浏览器之外地方（例如服务器和桌面应用上）使用的模块化技术。

## 规范

一个文件就是一个模块，拥有单独的作用域。

## 定义模块

使用 exports 或 modul.exports

    var myModul = (a, b)=>{
        return a+b;
    }
    module.exports = myModul;

## 使用模块

使用 require

    var mm = require('./myModul.js');
